Alt-text description

This interactive flashcard activity teaches beginning Java concepts from Chapters 1 and 2. Each card has a question on the front and an answer with a short Java or pseudocode example on the back. Topics include pseudocode, probability, special symbols, indentation, programming errors, variables, constants, Scanner input, and mathematical operations.
