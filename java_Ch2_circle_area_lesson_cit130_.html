Quick Reference — Key Java Concepts

Review these concepts used in your program

📦

import Statement

Brings external Java classes into your program. import java.util.Scanner; gives access to the Scanner class for reading input.

🏛️

Class Declaration

Every Java program is wrapped in a class. public class CircleArea {} defines the container for all your code.

🚀

main Method

Java programs begin execution at public static void main(String[] args). Without this signature, the program won't run.

⌨️

Scanner Object

Creates a bridge to the keyboard. scanner.nextDouble() reads a decimal number the user types into the console.

🔢

Constant Variable PI

final double PI = 3.1416; creates a value that should not change. Use PI * radius * radius to calculate the circle area.

🖨️

System.out.println

Prints text to the console and moves to the next line. Use System.out.print (no "ln") to stay on the same line.
